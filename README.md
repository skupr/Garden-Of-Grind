# Garden Of Grind Addon Mod
A drop-in helper mod for the garden of grind challenge run for GTNH.

Well... almost. If you want the quest pages, add a BQApi, https://github.com/ABKQPO/BetterQuestingAPI, of at least 1.1.0

If using this, make no other gog config changes. You do not need to modify Thaumcraft.cfg, Witchery.cfg, GregTech.cfg, or Hodgepodge. All changes are done internally by this addon.

Contains a questbook page (TODO), guideNH pages (TODO), and recipe changes based on chosen challengeMode (mostly done).

## Modes

So. Heard the word 'mode' and got excited? 

Indeed. We got modes! 

We have: Garden of Grind (the base version, no recipe changes), Skyblock (Some recipe and mechanic changes to make it closer to a skyblock version of GTNH), Garden of Grindless (lightens several of the harder grinds, unlocks some stuff earlier), and Questless Garden of Grind (disables quest rewards. Does nothing else.)

To toggle a mode, either do so through the GardenOfGrind config, or in-game through /gog mode <0~3>, the values corresponding to the order I gave here. You can also press tab to do the command by mode name. 

### Garden of Grind

Like I said, the base version. There have been no mechanic changes done besides disabling entity cramming.

### Skyblock

Has several recipe and mechanic changes, such as tossing gravel dust in water (lossy) or a cauldron (less lossy) for metals, centrifuging gravel and sand through several tiered recipes for metal ores, tossing silt into water to make clay dust, or using the new Thermite Bucket to make your first lava.

Combine the items, give it a spark (toss it in fire), and you better pick up the resulting bucket fast, or it might melt, too!

A source of an early sapling and an easy dirt recipe is pending, so just cheat them in for now.

### Garden of Grindless

Has a few of the changes made for Skyblock, such as gravel dust for metals, the most basic centrifuging recipes, and a skystone rock breaker recipe.

The outputs of the tengam and debris meteor have been buffed and the thermite bucket to lava recipe is present, overall lessening the grind. Might be a bit too much, will see.

### Questless Garden of Grind

The questless run is actually completely possible with no changes to the base game! ...in 2.9 at least. Should something happen, I'll add minimal changes to make it possible again. 

For now, it just disables the questbook rewards.
